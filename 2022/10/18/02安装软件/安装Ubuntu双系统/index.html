前言

一般有两种方法，一是使用虚拟机，二是安装双系统。使用虚拟机的好处是占用资源少，相对简单方便，不会碰到实装时出现的一些兼容性问题，使用户的注意力能更好地集中于使用 linux 本身，并且可以创建多个 linux 环境从而可以使用集群，来回切换 Windows 和 Ubuntu 比较方便。而安装双系统属于实装，会消耗更多硬件资源，会碰到很多兼容性方面的问题，锻炼使用者在实践中发现问题和解决问题的能力，美中不足的是，这种方法无法创建更多 linux 环境，因而不能使用集群操作。

如果是初次接触 linux 系统，建议在虚拟机里安装使用，这样和外部系统隔开，无论出现什么问题、无论如何折腾都不会影响到当前使用的系统，影响已有的工作环境。虚拟机如 vmware 或者开源的 virtualbox 都是很好的选择。如果已经对 linux 有一定了解了，其实装单系统还是双系统就纯看个人喜好和需求了，比如我个人在在虚拟机里用了大概一个多月的 ubuntu。

答案是先安装windows，有windows系统后，安装双系统才会更加方便，这个是基础，通常来说安装双系统是先安装windows，再安装linux，但是不能排除有时候只能先安装linux，再安装windows。例如新买的机器，里面只有linux系统。这种情况下安装会稍微复杂一点，最大的区别是为windows系统预留空间。
